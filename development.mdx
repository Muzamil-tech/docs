<Info>
  **Prerequisite**: Please install Node.js (version 19 or higher) before proceeding.
</Info>

<Info>
  **Prerequisite**: Please composer before proceeding.
</Info>

Follow these steps to install and run the system on your local:

**Step 1**: Install packages:

```bash composer
composer install
```

**Step 1**: Install dependencies:

<CodeGroup>
  ```bash npm
  npm i 
  ```
</CodeGroup>

**Step 3**: run the system

```bash
composer run dev
```

A local preview of your system will be available at `http://localhost:8000`.
